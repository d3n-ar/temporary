# Tutorial 

## Preliminary 
1. Download the gba file by clicking on it and clicking *view raw*
2. go to files and make a new folder called "GBAgames"
3. Move your gba file from Downloads to GBAgames folder.
4. follow the instructions depending on the emulator of your choice.

## Android - Pizza Boy

1. Get the apk file by clicking on it and install it.
2. Open it and select "Add ROMs folder" from the options.
3. Select GBAgames folder
4. It should automatically scan and find the game. If it doesn't, manually scan it.
5. Disable Linear Filtering by going to Settings -> Video
6. You can add icon and shortcut by holding the game if you want to.
7. Play!

## Android - Lemuroid 

1. For lemuroid to access it, go to GBAgames folder and select the gba file.
2. Download Lemuroid from Google Play Store.
3. Add GBAgames folder in Lemuroid.
4. Let it scan and play!

## Android & PC - Retroarch 

***Note:- This is heavily confusing for many, it's not recommended to use retroarch unless you know what you're doing.***
1. Download Retroarch from the liberto website.
2. Go to Online Updater -> Download Core
3. Install mGBA core. (VBA-m doesn't have guaranteed support)
4. Select Core from menu after downloading 
5. Load directory and find GBAgames folder
6. There Play the game

## PC - mGBA
1. Download mGBA from their website.
2. Open it and select the file from 'file' option.
3. Play!
